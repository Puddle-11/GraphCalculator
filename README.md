Instructions:

This project has 4 scripts.
Point manager: handles al the adding and removing of points from the board.
Camera Manager: Manages the mouse camera and UI
Line: Handles each line, this includes math for the end points, and controlling the arrow head
Point: Handles point math this includes, click events for adding lines, and drag events for mouse hovers.
